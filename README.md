# Basic-Image-Processing
Basic Image Handling and Processing Laboratory Exercise for CS124P

Contents
1. Annotation and Pixel Manipulation: Python script to select 10 points on an image and display their RGB values. 
2. RGB Image Slicing: A dynamic 180x180 RGB image generator (checkerboard pattern). 
3. Gray Transform and Histogram: Comparison of 4 different gray transforms and their resulting histograms.
